TP 2 : Entités Salle et Utilisateur avec Hibernate

- Description :
Ce projet est un TP réalisé dans le cadre du module Java/JPA.  
Il permet de gérer des Utilisateurs et des Salles en utilisant :JPA (Java Persistence API), Hibernate, Base de données H2 (en mémoire) ,Maven

-Technologies utilisées :
DK 8 ou supérieur installé , Maven installé , IntelliJ IDEA

- Fonctionnalités implémentées :
  CRUD complet pour Utilisateur
  CRUD complet pour Salle
  Recherche par email
  Recherche par disponibilité
  Recherche par capacité minimale
